Win32 Installation:
	1. Install QMM
	2. Place the nocrash_qmm.dll file in your mod directory (i.e. baseq3)
	3. Add "nocrash_qmm.dll" as an entry in the plugins list in qmm.ini/qmm.cfg/qmm.conf

Linux Installation:
	1. Install QMM
	2. Place the nocrash_qmm.so file in your mod directory (i.e. baseq3)
	3. Add "nocrash_qmm.so" as an entry in the plugins list in qmm.ini/qmm.cfg/qmm.conf
